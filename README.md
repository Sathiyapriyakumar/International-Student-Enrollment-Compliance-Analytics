#  International Student Enrollment & Compliance Analytics

## Overview
This project analyzes international student enrollment and compliance data using Applicant, Connect, and SEVIS datasets. It focuses on data cleaning, preprocessing, PostgreSQL integration, and interactive dashboard reporting to support data-driven decision-making.

---

## Features
- Data Cleaning & Validation
- Missing Value Analysis
- Duplicate Removal
- Data Standardization
- PostgreSQL Database Integration
- Compliance & Enrollment Analysis
- Interactive Dashboard Reporting

---

##  Technologies Used
- Python
- PostgreSQL
- SQL
- Power BI / Looker Studio
- Pandas
- Excel

---

##  Key Insights
- 6,893 student records analyzed
- 97.9% eligibility rate
- 34.7% I-20 completion rate
- 45.5% I-901 fee payment rate
- Identified compliance risks and enrollment bottlenecks

This repository contains project documentation, SQL scripts, Python scripts, and dashboard screenshots.

The original institutional datasets are **not included** due to confidentiality.
